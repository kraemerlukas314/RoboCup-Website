Hardware overview

  • 1x Arduino Uno
  • 1x Mindsensors EVShield
  • 1x LEGO touch sensor
  • 2x LEGO color sensor
  • 2x Pololu 20D (12V, 140PWM)
  • 1x L293D H-bridge
  • 1x 12V step up module
  • 1x Floureon 2S LiPo

Description

Our Robot is powered by an Arduino Uno and an EvShield by Mindsensors which allows us to use LEGO sensors without an Ev3.
Our 3d printed chassis holds two LEGO color sensors to follow the line and detect green turn signals. To detect the obstacle we mounted a LEGO touch sensor to our chassis which we can also use with the EVShield. For driving we use two Pololu 20D (12V, 140PWM) motors.
The holes you can see around the chassis are compatible with LEGO pieces so if we want to mount something to our robot afterwards we can just screw in some LEGO pieces.
To detect whether the turn signals are in front of or behind the black line as soon as the sensors detect green the robot drives a little bit backwards and forwards. If the robot detects any black during this the turn signal must be behind the black line and therefore ignored. You can find our source code on our GitHub repository.
